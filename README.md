# Seurat

Convert videos into ASCII art animations entirely in the browser.

## Features

- Runs 100% client-side - no server required
- Export as MP4 (H.264) or compressed ASCII format
- Customizable FPS, ASCII width, characters, and noise level
- Preserves colors from original video
- Built-in player with ASCII and dots render modes

## Usage

Visit [seurat.neiltthomas.com](https://seurat.neiltthomas.com) or run locally:

```bash
cd web
npm install
npm start
```
